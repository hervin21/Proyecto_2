HTML

El diseño responsivo se basa en HTML y CSS, una combinación de dos lenguajes que controlan el contenido y el diseño de una página en cualquier navegador web. El HTML controla principalmente la estructura, los elementos y el contenido de una página web. Por ejemplo, para añadir una imagen a una página web, tienes que usar código HTML como este:

img src="image.gif" alt="image" class=”full-width-img”

Puedes establecer una «clase» o un «id» que luego puedes atacar con el código CSS. También puede controlar propiedades clave como el alto y el ancho en HTML, pero esta ya no se considera la mejor opción. En cambio, CSS se utiliza para editar el diseño y la disposición de los elementos de la página HTML. El código CSS se puede incluir en la sección style de un documento HTML o como un archivo de hoja de estilo independiente. Por ejemplo, podríamos editar el ancho de todas las imágenes HTML en el nivel de elemento como este:

img {

width: 100%;

}

O podríamos apuntar a la clase específica «full-width-img» añadiendo un punto al frente.

.full-width-img {

width: 100%;

}

También puede controlar el diseño más allá de la altura, anchura y color. Usando CSS así es como haces que un diseño responda cuando lo combinas con una técnica llamada «media query».

MEDIA QUERIES

Las media query son una parte esencial de CSS3 que permiten que el contenido se represente para adaptarse a diversos factores, como el tamaño de la pantalla o la resolución. Funciona de manera similar a la cláusula "if" en algunos lenguajes de programación, principalmente verificando si la pantalla es lo suficientemente ancha o demasiado ancha antes de ejecutar el código correspondiente.

DISEÑO LIQUIDO

Un diseño líquido es una parte importante del diseño responsivo moderno. Anteriormente, podías establecer cada elemento HTML en un valor estático, como 600 píxeles. En cambio, los diseños flotantes dependen de valores dinámicos como porcentaje del ancho de la vista. Este método aumenta o disminuye dinámicamente los distintos tamaños del elemento contenedor según el tamaño de la pantalla.
